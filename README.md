# Random Password Generator

**Автор:** Гречуков Данил

## Описание

Графическое приложение (GUI) для генерации случайных паролей с настройкой:
- Длины (от 4 до 64 символов) через ползунок.
- Включения/выключения цифр, строчных и заглавных букв, спецсимволов.
- Просмотра истории сгенерированных паролей в таблице.
- Сохранения истории в файл `history.json` и автоматической загрузки при запуске.
- Копирования пароля в буфер обмена.
- Очистки истории.

Приложение написано на Python 3 с использованием стандартной библиотеки `tkinter`.

## Требования

- Python 3.6 или выше
- Стандартная библиотека (ничего дополнительно устанавливать не нужно)
